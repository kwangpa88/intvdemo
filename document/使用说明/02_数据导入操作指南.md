# 数据导入操作指南

## 1. 概述

数据导入脚本（`csvimport/importCsv.ts`）负责将 CSV 格式的原始 ROI 数据批量写入 SQLite 数据库（`app_roi.db`）。

---

## 2. CSV 文件说明

### 文件位置

```
backend/csvimport/app_roi_data.csv
```

### 文件格式

- 编码：UTF-8（无 BOM）
- 分隔符：英文逗号 `,`
- 首行：表头

### 表头定义

```
日期,app,出价类型,国家地区,应用安装.总次数,当日ROI,1日ROI,3日ROI,7日ROI,14日ROI,30日ROI,60日ROI,90日ROI
```

### 数据示例

```csv
日期,app,出价类型,国家地区,应用安装.总次数,当日ROI,1日ROI,3日ROI,7日ROI,14日ROI,30日ROI,60日ROI,90日ROI
2025-04-13(日),App-1,CPI,美国,4849,6.79%,14.24%,27.3%,64.44%,120.89%,214.65%,368.42%,413.81%
2025-04-13(日),App-2,CPI,美国,2180,4.19%,14.74%,19.54%,35.63%,64.57%,107.64%,186.08%,206.66%
```

---

## 3. 字段转换规则

| CSV 原始值 | 转换规则 | 数据库存储值 |
|---|---|---|
| `2025-04-13(日)` | 去除 `(X)` 星期后缀 | `2025-04-13` |
| `4849` | `parseInt()` | `4849` |
| `6.79%` | 去除 `%` + `parseFloat()` | `6.79` |
| `100%` | 同上 | `100.0` |
| `0%` | 同上 | `0.0` |

---

## 4. 执行导入

### 基本导入命令

```bash
cd backend
npm run import:csv
```

### 预期输出

```
[DB] app_roi 테이블 준비 완료 (파일: app_roi.db)
(node:XXXXX) ExperimentalWarning: SQLite is an experimental feature...
[CSV Import] 완료: 910건 삽입 → app_roi.db
```

---

## 5. 重置数据库后重新导入

若需要清空数据库并重新导入（如 CSV 文件内容已更新）：

```bash
# 步骤 1：销毁并重建表（启动服务时触发）
cd backend
DROP_TABLE=true npm run dev
# 看到 "[DB] app_roi 테이블 준비 완료" 后关闭服务 (Ctrl+C)

# 步骤 2：重新导入 CSV
npm run import:csv
```

> ⚠️ **警告**：`DROP_TABLE=true` 会删除所有现有数据，操作前请确认。

---

## 6. 替换 CSV 文件

如需导入新的数据文件：

1. 将新 CSV 文件命名为 `app_roi_data.csv`，放置于 `backend/csvimport/` 目录
2. 确保表头格式与上述定义一致
3. 执行重置并重新导入（参考第 5 节）

或修改 `importCsv.ts` 中的文件路径常量 `CSV_PATH`：

```typescript
// backend/csvimport/importCsv.ts
const CSV_PATH = path.resolve(__dirname, 'app_roi_data.csv')  // 修改此处
```

## 8. 导入脚本技术说明

脚本文件：`backend/csvimport/importCsv.ts`

### 关键逻辑

```typescript
// 1. 使用 mapHeaders 在解析时将中文表头映射为英文列名
csvParser({
  mapHeaders: ({ header }) => HEADER_MAP[header.trim()] ?? header.trim()
})

// 2. 批量插入使用 PreparedStatement（性能优化）
const insert = db.prepare(`INSERT INTO app_roi (...) VALUES (?, ?, ...)`)

// 3. 逐行 insert.run()（SQLite 同步 API）
for (const row of rows) {
  insert.run(parseDate(row.date), row.app, ...)
}
```

### 表头映射（`HEADER_MAP`）

| CSV 表头（Unicode） | 数据库列名 |
|---|---|
| `日期` | `date` |
| `app` | `app` |
| `出价类型` | `bid_type` |
| `国家地区` | `country_region` |
| `应用安装.总次数` | `total_installs` |
| `当日ROI` | `day0_roi` |
| `1日ROI` | `day1_roi` |
| `3日ROI` | `day3_roi` |
| `7日ROI` | `day7_roi` |
| `14日ROI` | `day14_roi` |
| `30日ROI` | `day30_roi` |
| `60日ROI` | `day60_roi` |
| `90日ROI` | `day90_roi` |
