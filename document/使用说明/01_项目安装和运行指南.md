# 项目安装和运行指南

## 1. 环境要求

| 工具 | 最低版本 | 说明 |
|---|---|---|
| Node.js | **22.5.0+** | 内置 `node:sqlite` 模块从 22.5.0 开始提供（实验性） |
| npm | 9+ | 随 Node.js 一起安装 |
| 操作系统 | Windows / macOS / Linux | 无限制 |

> 验证 Node.js 版本：`node -v`

---

## 2. 获取项目代码

如使用 Git：

```bash
git clone <仓库地址>
cd intvdemo
```

或直接解压项目压缩包至目标目录。

---

## 3. 安装后端依赖

```bash
cd backend
npm install
```

安装的主要包：

| 包 | 说明 |
|---|---|
| `express` | HTTP 服务器 |
| `cors` | 跨域处理 |
| `csv-parser` | CSV 文件解析 |
| `ts-node-dev` | TypeScript 热重载开发工具 |
| `@types/node` | Node.js 类型声明（含 `node:sqlite`） |

---

## 4. 安装前端依赖

```bash
cd frontend
npm install
```

---

## 5. 初始化数据库并导入数据

首次运行前，需要将 CSV 数据导入 SQLite 数据库：

```bash
# 在 backend 目录下执行
cd backend
npm run import:csv
```

执行成功后会输出：

```
[DB] app_roi 테이블 준비 완료 (파일: app_roi.db)
[CSV Import] 완료: 910건 삽입 → app_roi.db
```

同时在 `backend/` 目录下生成 `app_roi.db` 文件。

> **注意**：再次执行 `import:csv` 会在现有数据基础上继续追加，若需重置数据库，请参考第 6 节。

---

## 6. 启动后端服务

```bash
cd backend
npm run dev
```

启动成功后输出：

```
[DB] app_roi 테이블 준비 완료 (파일: app_roi.db)
Backend server running at http://localhost:4000
```

后端监听 **`http://localhost:4000`**。

### 可用脚本

| 命令 | 说明 |
|---|---|
| `npm run dev` | 开发模式启动（ts-node-dev，支持热重载） |
| `npm run build` | 编译 TypeScript 至 `dist/` |
| `npm run start` | 生产模式启动（需先 build） |
| `npm run import:csv` | 批量导入 CSV 数据至数据库 |

### 环境变量

| 变量 | 值 | 说明 |
|---|---|---|
| `PORT` | 整数（默认 `4000`） | 后端监听端口 |
| `DROP_TABLE` | `true` | 启动时销毁并重建 `app_roi` 表（**数据会丢失**） |

```bash
# 示例：重置数据库并启动
DROP_TABLE=true npm run dev
```

---

## 7. 启动前端服务

**新开一个终端窗口**：

```bash
cd frontend
npm run dev
```

启动成功后输出：

```
▲ Next.js 14.1.0
- Local: http://localhost:3000
✓ Ready in 6s
```

前端监听 **`http://localhost:3000`**。

---

## 8. 访问应用

在浏览器中打开：

```
http://localhost:3000
```

页面会自动重定向至 `http://localhost:3000/dashboard`，展示 ROI 趋势大盘。

---

## 9. 同时运行两个服务（Windows PowerShell）

可以在两个独立的 PowerShell 窗口中分别运行，也可以使用以下方式：

```powershell
# 终端 1 — 后端
Set-Location d:\work\intvdemo\backend ; npm run dev

# 终端 2 — 前端
Set-Location d:\work\intvdemo\frontend ; npm run dev
```

## 11. 常见问题

| 问题 | 解决方案 |
|---|---|
| `Cannot find module 'node:sqlite'` | 升级 Node.js 至 22.5.0 以上，并升级 `@types/node` 至 latest |
| `EADDRINUSE: address already in use :::4000` | 关闭占用 4000 端口的进程，或修改 `PORT` 环境变量 |
| 前端页面空白 / API 调用失败 | 确认后端已启动，且运行在 `localhost:4000` |
| 数据库查询结果为空 | 确认已执行 `npm run import:csv` |
